# Sharing & Collaboration Guide

Nimbus makes it easy to work on files and documents together. This guide explains how to share folders with the right level of access, create share links, request files from people outside Nimbus, hand off ownership, and collaborate in Nimbus Paper.

You can share from the Web, the Desktop app (Windows, macOS, Linux), or Mobile (iOS, Android).

## Shared folders and roles

Shared folders let you bring other people into a folder so everyone works from the same files. Each person you add is assigned a role that controls what they can do.

### Roles explained

- **Viewer** — Can open and view files, but cannot make changes.
- **Commenter** — Can view files and leave comments, but cannot edit content.
- **Editor** — Can view, comment, and edit files in the folder.
- **Manager** — Has full control, including editing files, adding or removing people, and changing other members' roles.

Choose the lowest role that still lets someone do their job. You can change a person's role at any time as your collaboration evolves.

### Share a folder

1. Locate the folder you want to share.
2. Select **Share**.
3. Enter the email addresses of the people you want to add.
4. Choose a role (Viewer, Commenter, Editor, or Manager) for the people you're inviting.
5. Send the invitation.

### Change a role or remove someone

1. Open the folder and go to **Manage access**.
2. Find the person whose access you want to update.
3. Change their role, or remove them from the folder.

## Share links

Share links are a quick way to give someone access without adding them to a folder. Anyone with the link can open the shared content.

Links can be set to:

- **View** — recipients can see the file or folder.
- **Edit** — recipients can make changes.

To create a share link:

1. Select the file or folder you want to share.
2. Choose **Create link** (or **Copy link**).
3. Set the link to **View** or **Edit**.
4. Copy the link and send it to your recipients.

Every plan includes basic sharing links. Keep in mind that shared-link traffic has a daily limit: 20 GB/day on Free, 1 TB/day on Plus and Family, and 4 TB/day on Business.

### Link passwords and expiry (paid plans)

On Plus, Family, and Business plans, you can add extra protection to your share links:

- **Password protection** — require a password before anyone can open the link.
- **Expiry dates** — set a date after which the link stops working automatically.

These options are useful when you're sharing sensitive material or only want a link to be active for a limited time. Configure them when you create the link, or update them later from the link's settings.

## File requests

A file request lets other people upload files into one of your folders **without needing a Nimbus account**. This is ideal for collecting documents, photos, or submissions from clients, vendors, or teammates.

1. Choose the folder where you want uploaded files to land.
2. Create a **file request** for that folder.
3. Share the request link with the people you want to collect files from.

People who use the link can upload to your folder, but they can't see the other files it contains.

Remember that any files uploaded to you count toward your storage.

## Transferring ownership

If you need to hand a folder off to someone else — for example, when changing roles or leaving a project — you can transfer ownership.

1. Open the folder and go to **Manage access**.
2. Locate the person you want to make the new owner.
3. Transfer ownership to them.

Because Nimbus accounts cannot be merged, sharing a folder (and transferring ownership when appropriate) is also the recommended way to move files between two accounts.

## Nimbus Paper

Nimbus Paper is built for collaborative documents. Multiple people can work in the same doc, and Paper includes tools that keep everyone aligned:

- **Comments** — leave feedback directly in the document.
- **@mentions** — tag a collaborator to bring something to their attention.

Paper docs are shared using the same roles and links described above, so you can decide whether someone can view, comment on, or edit a document. Note that Nimbus Paper docs share your account's storage along with your files and backups.

## Comments and version history

Beyond Paper, you can also add **comments** to files and rely on **version history** to track changes over time. If something goes wrong, you can return to an earlier version. Version history is retained for 30 days on Free, Plus, and Family plans, and 365 days on Business.

Deleted shared files can be recovered from Trash within the same retention window.

## Tips for smooth collaboration

- Match each person's role to what they actually need to do.
- Use password-protected and expiring links (on paid plans) for anything sensitive.
- Use file requests to collect uploads from people who don't have Nimbus.
- Transfer folder ownership when responsibilities change, rather than trying to merge accounts.

If you run into trouble while sharing, you can reach support by email and chat (Free, Plus, and Family), priority support (Business), or phone and a dedicated Customer Success Manager (Enterprise). You can also check current service status at status.nimbus.example.
